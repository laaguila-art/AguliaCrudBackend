# Agulia CRUD Frontend

A simple React frontend for the Agulia CRUD Backend with authentication and position management.

## Features

-  User Authentication (Login & Register)
-  CRUD Operations for Positions
-  Clean and Modern UI
-  Protected Routes
-  Responsive Design

## Backend API

The frontend connects to: https://aguliacrudbackend.onrender.com

## Local Development

1. Install dependencies:
npm install

2. Run the development server:
npm run dev

3. Open your browser at http://localhost:5173

## Deploy to Vercel

### Option 1: Using Vercel CLI

1. Install Vercel CLI:
npm i -g vercel

2. Deploy:
vercel

### Option 2: Using Vercel Dashboard

1. Push your code to GitHub
2. Go to https://vercel.com/dashboard
3. Click 'New Project'
4. Import your GitHub repository
5. Vercel will automatically detect it's a Vite project
6. Click 'Deploy'

## Usage

1. Register: Create a new account
2. Login: Sign in with your credentials
3. Manage Positions: View, Create, Edit, and Delete positions

## Technology Stack

- React 18
- Vite
- React Router v6
- Axios
- CSS3
